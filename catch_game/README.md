# 🎮 Catch The Ball Game (Python + Pygame)

## 📌 Project Description
Catch The Ball is a mobile-style 2D game developed using Python and Pygame.
The player controls a basket to catch falling balls.

## 🕹️ Features
- Continuous gameplay
- Score & level system
- Restart and exit option
- Keyboard controls

## 🛠️ Technologies Used
- Python 3
- Pygame
- VS Code

## ▶ How to Run the Game
1. Install Python
2. Install pygame:
   pip install pygame
3. Run:
   python main.py
